**Student X**

Welcome to Student X — a platform by students, for students, designed to help them discover and organize opportunities for personal, academic, and professional development. The website gathers initiatives from various organizations, making it easy for students to stay informed on events, programs, workshops, and other growth opportunities. Student X is fully accessible on mobile and tablet devices.

Hosted on GitHub Pages and maintained by a small team, Student X is a non-commercial, public-access platform developed by Rafael Faustino, Tomás Sousa, and Miguel Abelho.

**Key Features**

    Comprehensive Event Calendar: View and add events to personal calendars.
    A creative calendar layout that displays events according to the specific day and time they occur.
    Mobile & Tablet Friendly: Optimized design for various devices.
    Updated Information: Regularly updated with new opportunities from partnered organizations.
    Non-Editable Public Access: Publicly available for viewing only, with all rights reserved by the authors.

**Content Overview**

This repository contains all the necessary files for the Student X website, organized as follows:

    Root Files
        index.html - Main entry point of the website.
        CNAME - Custom domain configuration file for GitHub Pages.
        favicon.ico - Favicon for the website.
        LICENSE.md - Licensing terms for the repository.
        robots.txt - Directs web crawlers and specifies what content to index.
        sitemap.xml - Sitemap for SEO and improved navigation by search engines.

    Page Directories
        contacto/ - Contains index.html for the Contact Us page.
        oportunidades/ - Contains index.html for the Opportunities page.
        Política de Privacidade e Cookies/ - Contains index.html for the Privacy and Cookie Policy.
        Termos e Serviços/ - Contains index.html for the Terms of Service.

    CSS (css/)
        Contains custom stylesheets for various pages:
            contact us.css - Styling for the Contact Us page.
            oportunidades.css - Styling for the Opportunities page.
            politica de privacidade e cookies.css - Styling for Privacy and Cookie Policy.
            styles.css - General styles for the website.
            termos e serviços.css - Styling for Terms of Service page.

    JavaScript (js/)
        calendar.js - Manages the interactive calendar.
        cookie-banner.js - Controls cookie banner display and user interaction.
        script.js - Main script with functionalities for general interactivity.

    JSON (json/)
        events.json - JSON file that dynamically populates events on the site.

    Events Assets (events/)
        Contains images used to promote events from third-party companies:
            image1.png, image1logo.png, image2.png, etc.
        Note: These assets are used strictly for promotional purposes with respective rights reserved by their owners.

    Images (images/)
        Various images and graphical elements used across the site:
            Team member photos (abelho.jpg, rafael.jpg, tomas.jpg).
            Icons and UI elements (5stars.png, instagram.png, linkedin.png, etc.).
            Branding and logo assets (logoStudentX.png, logos.png, etc.).

    Videos (videos/)
        slogan.mp4 - Slogan video for promotional use.
        videoP2.mp4 - Secondary video for page content.

**Technologies Used**

    HTML, CSS, JavaScript: Core technologies used for building the website.
    JSON: Used to dynamically populate event information and carousel content.
    GitHub Pages: Hosts the website, enabling free public access.
